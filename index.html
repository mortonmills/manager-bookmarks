Browser Bookmarks Merger

When you have an excess of bookmarks, the bookmarks merger can remove duplicates and restructure your bookmarks.
There are 3 options.
The File option outputs one file listing all of your bookmark files as folders.
The Folder option outputs all folders and their children into one file.
The Line option outputs a list of each individual bookmark into one file.
